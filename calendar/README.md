## 目录结构

demo 用vue-cli 的webpack构建的，加了publish的文件,在发布npm的时候

```bash
　|--dist　build生成的目录
　|--doc   展示图片
  |--lib   publish 之后的文件，发布到npm用
　|--src
　　　|--assets 资源
　　　|--components
　　　　　|--Picker    日期组件
　　　　　|--PickerItem  滚动的子组件
      |--modules 插件的js
　　　|--css
　　　|App.vue   入口
　　　|main.js
```

## 组件使用

>安装

```bash
npm i vue2-datepick --save
```

>初始化

```js
import Calendar from 'vue2-datepick';
Vue.use(Calendar);
```

>使用

```js
 <script>

 export default {
   name: 'App',
   data(){
       return{
           date:'',
           date2:'2010-3-2'
         }
   },
   methods:{
     setDate(){

       this.$picker.show({
         type:'datePicker',
         onOk: (date) =>{
           this.date = date
         }
       });

     },
     setDate2(){

       this.$picker.show({
         type:'datePicker',
         date:this.date2,  //初始化时间
         endTime:'2020-02-11',  //截至时间
         startTime:'2010-02-11',  //开始时间
         onOk:(e)=>{
           this.date2 = e;
         },

       })

     },

   },

 }
 </script>




```
